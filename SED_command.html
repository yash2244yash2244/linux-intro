SED command

    - SED stands Stream Editor.
    - With the help of SED command we can perform below actions/operations.
    1. Print a particular line or multiple line
    2. Delete a particular line or multiple line
    3. Replace or subtitute one word with another word.
    4. Insert data before any line
    5. Insert data After any line
    -Syntax of sed command
      sed option 'action' file_name
