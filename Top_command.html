top command

    top command is not used to display top lines from a file.
    top command is used to display CPU utilization
    How to use top command?
               Ans: simply type top and press enter
    Ones top command get executed then in the out put below information get display
    1. Current time
    2. system up time (How long system is running)
    3. Number of process initiated
    4. Running process count
    5. Sleep process count
    6. Stoped process count
    7. Zombie process count
    8. % CPU usage
    9. % memory usage
    10. All initiated process ID (PID)
    11. Who has started process (owner of process)
    12. Process priority (PR)
    13. Process status (s means sleep, R means Running, Z-means Zombie)
    Note: To exit from top command we need press ctrl+z
