Creation of Link files

    There are two types of link files

    1. Soft link
    2. Hard link

1.Soft link file
    -- soft link is a pointer to another file.
    -- It is just like windows short cut.
    -- It is also called symbolic link.

    -- How to create a soft link?
       Ans: To create a soft link we need to use 'ln' command with -s option

    --syntax : to create a soft link
       ln -s source_file_name soft_link_name

    --Example: create a soft link file for Dumy.txt
       Ans: ln -s Dumy.txt Dumy_soft_link
    Important conclusion about soft link
    1. Original file and soft link file don't have same time stamp
    2. Original file and soft link file don't have same size
    3. Original file and soft link file don't have same inode number
    4. If we delete soft link file then there is no impact on original file.
    5. If we delete original file then there is impact on soft link
