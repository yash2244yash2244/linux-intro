Characteristics or features of Unix

  1. It is freeware software.

      2. i)if you want to use this os then no any need to pay


  2. It is open source software.

      3. i)source code is open.
      ii)if you are not satisfied with given features then you can modify it


  3. It is multi-user software.
      4. i)multiusers can use this software at a time

  4. It is multi-tasking software.

  5. It is user friendly and support for both
      6. i) CLI (command line interface)
      ii) GUI (graphical user interface)

  6. It is more secure than other Windows OS.

  7. It is more powerful than other operating system.
