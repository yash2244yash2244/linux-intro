tar command

- tar stands Tape Archieve
- tar command is used to group multiple files or multiple directory into single file.
- Syntax to Create archieve file of a single file
      tar option archieve_file_name.tar source_file_name
    Here while creating archieve file we use option -cvf
      c stands for create
      v stands for verbose
      f stands for forcefully

Note: Use .tar extension manualy to the archieve file name.
- Syntax to Create archieve file of a directory
      tar option archieve_file_name.tar source_dir_name
Note: while creating archieve file of a directory use .tar extension manualy
- Syntax to create archieve file of multiple files.
      tar -cvf f123.tar f1.txt f2.txt f3.txt
Note: source file nemes should be seperated by space


Extract content of archieve file

- To extract the content of an archieve file tar command is used with -xvf option
      tar option archieve_file_name
- While extracting tar file content use -xvf option
      x stands for extract
-Example: Extract f123.tar file content
      tar -xvf f123.tar
