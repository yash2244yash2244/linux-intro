       generating reports

Q. When we should go with awk command?
Ans: If there is irregular seperator and we cannot fix the delimeter to cut
        data from file then we should go with awk command

Synatax of awk command
awk <option> 'action' file_name
some important points about awk
1) $ symbol is used to specify field (column)
2) -F option is used to specify delimeter


- First Create a file
If we want to save/redirect any command output into a file
   command_name > File_name
I want to save df command ouput into a file
Ans: df -h > cutdemo.txt


Example 1: seperate the words on the bsis of space and print first field
using awk command
Syntax:    awk '{print $1}' cutdemo.txt
Here we have not used option -F to specify delimeter because
default delimeter is space.

You must specify action in single quote.
Here $ is used to specify column

Note: in awk default delimeter is space. i.e- If you are not specifying any
delimeter then bydefault it is considered as space


Example 2: seperate the words on the basis of space and print second field
using awk command
Syntax:    awk '{print $2}' cutdemo.txt


Example 3: seperate the words on the basis of space and print second field
using awk command
Syntax:    awk '{print $4}' cutdemo.txt


Example 4: seperate the words on the basis of space and print first field, third field
and sixth field
Syntax:    awk '{print $1,$3,$6}' cutdemo.txt


Example 5: seperate the words on the basis of colon operator and print
first field
Syntax:    awk -F':' '{print $1}' cutdemo.txt


Example 6: seperate the words on the basis of colon operator and print second and third field
Syntax:    awk -F':' '{print $2,$3}' cutdemo.txt
