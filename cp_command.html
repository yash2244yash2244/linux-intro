cp command

    cp stands for copy
    Purpose : to copy a file or directory
    Use case 1 : to copy from one file to another file
    Syntax for use case 1
       cp source_file_name distination_file_name

    # cp F1.txt F2.txt
    1. Total content from a F1.txt will be copied to F2.txt
    2. If destination file is not available then this command will create distination file
    3. If destination file is available and contains some data then this data will be override with source file dara
    4. Surce file must be available, if source file is not available then we will get error


    #To copy F1.txt from directory A to directory B
      Syntax:-  cp A/F1.txt B/F2.txt

    # To copy all files from one directory to another directory
      Syntax :
               cp source_directory_name/* destination_ directory_name
      Example :   cp A/* B

    #To copy from different location and pest to different location
      Syntax:
           cp source_directory-name/* destination_directory_name
      Example :   cp A/B/C/* X/Y/Z

    To copy total directory to another directory
      Syntax :
           cp <option> source_directory_name destination_directiory_name
      Example:  cp -r A BB

Note: whenever we are copying one directory to another directory then compulsary we need to use -r
          here -r means recursive
