#Create zip file



- To create a zip file we use gzip command
- Syntax:
           gzip input_file_name
- Example: Create a zip file of NumList.txt
           gzip NumList.txt
- Ones command get executed we will get zip file and name of zip file will be NumList.txt.gz
- Note: Zip file has extension .gz

    # Creating zip files of multiple files at a time
    - Syntax:
               gzip file_name file2_name file3_name
    Example :
               gzip f1.txt f2.txt f3.txt
    - Ones above command get executed then in the output we
      will get 3 zip files and name of zip file will be
      F1.txt.gz, f2.txt.gz and f3.txt.gz


#Creating of zip file of Directory


- By default we can't create zip file of a directory using gzip command.
- If we want to create a zip file of a directory then first we need convert that directory into
  archieve file and then convert that archieve file into into zip file.
- Example: Suppose I have a directory 'DirE' and I want to create a zip file of DirE directory

    Step1: Create an archieve file of DirE
             tar -cvf DirE.tar DirE
    ones this command get executed we will get
                      DirE.tar file


    Step 2: Create zip file of that tar file as below
             gzip DirE.tar
    ones this command get executed then we will get
                      DirE.tar.gz
