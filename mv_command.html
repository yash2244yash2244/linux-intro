mv command

    about mv command
    mv stands for move
    mv is command is used for two purpose
    1. rename a file or directory
    2. cut and paste

1. Rename a file or directory using mv command

    Syntax to rename a file
         mvsource_file_name new_file_name
    Example: file Dumy2.txt present in DirD directory rename Dumy2.txt as Dumy.txt
    Ans:    mv DirD/Dumy2.txt DirD/Dumy

    Renaming of Directories

      Syntax:   mv  old_directory_name_or_path  new_directory_name_or_path
      Example:    mv A AAA
         Directory A will be renamed to AAA

2. cut and paste operation using mv command.

    We can cut a file from one location and paste into another location using mv command
    We can cut all files and sub-directory from one location and paste into another location using mv
    We can cut a whole directory from one location and paste another in another location
    Moving is nothing but cut and paste operation.
    Below are some use cases to cut and paste operation using mv command.

  1. Example 1: Cut a Dumy.txt file from DirD directory and paste in DirB directory.

      2. Syntax:  mv DirD/Dumy.txt DirB/
                              or
      mv DirD/Dumy.txt DirB/Dumy1.txt

Below are some use cases to cut and paste operation using mv command:


    Use Case 1: cut a file and paste
      Syntax: mv source_file_name destination_file_path

    Example: Cut a Dumy.tx file from DirD directory and paste in DirB directory

    Ans:      mv DirD/Dumy.txt DirB/

                                     OR

    mv DirD/Dumy.txt DirB/Dumy1.txt


    Use Case2: cut all files and sub-directories
      Syntax:   mv source_dir_path/* Distination_dir_path

    Example: Cut all files and subdirectory from DirD and paste in DirE Directory

    Ans:    mv DirD/* DirE


    Use Case 3: cut a whole directory and paste
      Syntax:   mv source_dir_path destination_dir_path

    Example: cut a whole DirD and past in DirB

    Ans:   mv DirD DirB
