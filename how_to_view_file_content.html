How to view the content of file or to display content of a file on screen.

    To display file content on screen we have below commands
      cat command
      tac command
      more command
      less command
      head command
      tail command
      reve command

by using cat command

    syntax:  cat -n file_name
    Example:  cat -n /Abc.text

by using head command

    Use: If we want to display top lines from a file then use head command
    Syntax:   head -n file_name
    Here -n indicate number of top lines do you want to display
    Example:  head -5 Abc.txt

by using tail command

    Use: If we want to display bottom lines from a file then use tail command
    syntax: tail -n file_name
    Here -n indicate how many bottom lines do you want to display.
    Example tail -5 Abc.txt

tac command :

    First line will become last line and last line will become first

more command

    It is used to display the content of a file page by page
    syntax:   more file_name
    Example:  more f1.txt
    When this command is executed it will display first page
      -  enter key to view content line by line
      -  space bar to view next page or press d key
      - q is used to quite/exit
    Note: with the help of more command we can navigate only in forward direction

less command

    less command is used to display file content page by page
    with help of less command we navigate forward as well as backword direction
    Syntax:   lees file_name
      example:  less /etc/passwd ones command get executed it will display first page of file
        1 If we want to navigate line by line press enter
        2 If we want to go on next page then press d or space bar
        3 If we want to go on previous page then press b
        4 If we want to exit then press q
