find command


# find command
- find command is used to search a file or directory in a file system
- find command searches file or directory recursively

- syntax:
     find search_location what_to_find option file_or_dir_name

- search location : where we want to find a file?
- If we want to search a file in current working path then use dot(.)
- If we want to search a file in a particular location then specify
    that location or that path
- Example suppose, I want to search a file f1.txt at A/B/C then
specify that path in search location

What to find : this part is an optional
- Here we need to specify type
- type means either file or directory
   - If we don't specify type then by default
find command searches for both-file and directory
- If we want to specify type then use -type and then use f for a file d for directory
    - type f means search only for files
    - type d means search only for directory

option : option means on which basis we want to find a file
-> we have different options
  1. -name : find a file or directory on the basis of name
  2. -user : find a file or directory on the basis of user name
  3. -group : find a file or directory on the basis of group name
  4. -perm : find a file or directory on the basis of permission
  5. -size : find a file or directory on the basis of size

      6. To specify size use below symbols/characters
      1. M -> MB
      2. G -> GB
      3. c -> byte
      4. + -> Greater than
      5. - (minus) -> less than

  6. To specify time or find a file on the basis of time use below options
    1. -ctime -> created time
    2. -mtimr -> modified time
    3. -atime -> Acess time
      4. Note: Here we need to specify values in days
    4. -cmin -> created time in minute
    5. -mmin -> modified time in minute
    6. -amin -> access time in minute
  7. -empty: To find an empty file or directory


Example 1: search f1.txt file in current working path
Syntax:find . -type f -name f1.txt

Example2 : If we type find and press enter then I will display all files and directory present at current location

Example 3: If we type find -type f
It will list out only files from current working path

Example 4: If we type find -type d
It will list out only directory from a current path

Example 5: List out all .txt files from current working path
Syntax:find .-type f -name '*.txt'

Example 6: list out all .log files from current working path
Syntax:find . -type f -name '*.log'

Example 7: List out all files and directory whose name start with D from current working location
Syntax:find . -name 'D*'

Example 8: Find only files whose name start with D from current location
Syntax:find . -type f -name D*

Example 9: Find only Directory whose name start with D from current location
Syntax:find . -type d -name 'D*'

Example10: Find files or directory whose name end with 't' from current location
Syntax:find . -name '*t'

Example11: Find fiels or directory whose name contains 'g'
Syntax:find . -name '*g*'

Example12: Find files or directory whose second letter is 'a'
Syntax:find . -name '?a*'

B. finding files and directory on the basis of user name
syntax: find location type -user user_name

Example:: find files and directory which belongs to rrd user
Syntax: find . -type f -user rrd
find . -type d -user rrd
find . -user rrd


c. find files and directory on the permission basis

Example:: find only files from current location where permission is 777
Syntax: find . -type f -perm 777

Example: find only files from current location whose permission is not 777
Syntax: find . -type f ! -perm 777
