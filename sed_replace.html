Action 5: subtitute

    - Replace one word with another word - For subtitute we use s character
    - Syntax:  sed 's/source_word/target_word/' File_name

    Example: Find 'one' word in seddemo2.txt file and replace with '001'
                 sed 's/one/001/' seddemo2.txt
    Original File Subtitute file

    Note: by default subtitute take place only for first occurance If we want to replace all occurances then use g ( g for global)

    Example: find 'One' word form seddemo2.txt file and replace only second occurance with 0001
                 sed 's/One/0001/g' seddemo2.txt

    Here g means global

    Example: find 'One' word form seddemo2.txt file and replace only second occurance with 0001
                 sed 's/One/0001/2g' seddemo2.txt

    Here 2g means second occurance

    Example: find one word from seddemo2.txt from 4th line and replace only second occurance
                 sed '4s/One/0001/2g' seddemo2.txt


    Example: Find One word from seddemo2.txt from line number 2 to 5 and replace all the occurance with 0001
                 sed '2,5s/One/0001/g' seddemo2.txt


    Example: Find 'Three' word from seddemo2.txt by ingoring case replace with 0003.
                 sed 's/Three/0003/i' seddemo2.txt


    Example: Replacement for all occurance and ignore the case
                 sed 's/Three/0003/gi' seddemo2.txt


    Example:Add the # to start of every line
                 sed 's/^/#' seddemo2.txt

    Example: Add # symbol to every line and save the out in another file
                 sed 's/^/#/' seddemo2.txt > sampleout.txt
    Note : By default sed command changes take place only on screen not in actual file.
    If we want to reflect changes in a file then use -i option
                 sed -i 's/^#//' seddemo2.txt
    Ones command get executed then ouput will not display on screen
    it will reflect in the file
    i.e Actual file content get modified
