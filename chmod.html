chmod command

If we fire ls -l command then we will get output in below format
Total ouput is divided into 7 column
A B C D E F G
↓ ↓ ↓ ↓ ↓ ↓ ↓

A: This column give information about file type and file permission
B: This column give information about number of blocks allocated to file 1 Block= 1024 byte
C: Owner name (owner - who has created that file?)
D: Group name (That file belongs to which group?)
E: This column give information about file size
F: This column give information about file creation or file modification date
G: file or Directory name


We can change below thing related to file
  1. We can change file permission by using chmod command
  2. we can change file owner name by using chown command
  3. We can change file group name by using chgrp command
  4. We can change file time stamp by using touch command
  5. We can change file name using mv command


1. Change file permission or change file mode - Every file has three modes
    a. Read mode
    b. write mode
    c. execute mode

- Here mode means which operation we can perform on a file .
- Read mode is indicated by letter 'r'
- Write mode is indicated by letter 'w'
- Execute mode is indicated by letter 'x'
when we execute ls -l command then we will get long information
about file. in the ouput first column give information about file
type and file permission

- There are total 10 character in every row
- Every row First character indicate file type
- If file is normal then it is hypen -
- If file is a directory then it is d
- If file is a soft link then it is l
- If file is socket file then it is s
- Remaining 9 character are divided into three group and
these character talks about file permission
- Each group contains 3 character
rwx -- owner file permission (group1)
rwx -- Group file permission (group2)
rwx -- other user file permission (group3)


Q. How to change file permission?
-> To change file permission we need to use chmod command
-> syntax chmod file_permission_numric_value file_name
-> File permission numeric value is a always 3 digit nuber
-> first digit is for owner permission
-> second digit is for group permission
-> third digit is for other user permission
We calculate numeric value using below table
Operation Operation symbol Numberic Value
Read r 4
Write w 2
Execute x 1

    owner digit = 4+2+1
