ls command:

    ls stands for list
    Purpose this command is used to list out files and directories.
    Syntax:    ls option directory_name
                     ls option dire_path

    Example 1: ls
      --If we execute ls command without options and directory name or
      --directory path then it will list out all files and directories
      --name from current working path.

    Example 2: ls command with -a option
      ls -a
      --Here -a means all.
      --It will list out all files and directory from current working path
      --including hidden files and directories
      --Hidden File or Hidden Directory means - The file or
      --Directory whose name start with dot

    Example 3: ls command with -A option
      ls -A
      --Here -A means Almost all.
      --It will list out all files and directories including
      --single dot but not double dot.

    Example 4: ls command with -r option.
      --by default ls command display file name and directories
      --names in ascending order (A,B,...Z, a,b,c,...z)
      --ls -r : It will display file names and directory in reverse of
      --alphabetical Order (z,y,x....a,Z,Y,X,...A)

    Example 5: ls command with -s option
      Here -s means size.
      --ls -s : it will display file names along with its size.
      --If we want size in human readable format then use -h option
      --with -s as below
      --ls -sh


    Example 6: ls command with -t option
      --Here -t means time.
      --ls -t : Display file name as per date of creation.
      --The file which is created first means older file get display
      --first and new file or recently created file name get display at least

    Example 7: ls command with -l option
      --Here -l means long information.
      --ls -l command out put get display in 7 colums as below


      -- First column give information about file type and its permission.


      -- In first column there are total 10 characters.
      --Out of 10 first character always talks about file type.

      -- Remaining 9 characters from first column give information
          about permission


    --First Character tells file type
    1. If first character is  -  then it is normal file
    2. If first character is  d  then it is directory
    3. If first character is  l  then it is link file
    4. If first character is  c  then it is character file
    5. If first character is  b  then it is block special files
    6. If first character is  s  then it is socket file

    --Remaining 9 characters from first column give information about permission.
      first 3 character group give information about owner permission
      second 3 character group give information about group permission
      last 3 character group give information about other user permission
      remaining 9 characters are divided into 3 group and each group contains 3 character.
             rwx      r-x     r-x Here r,w,x give the information about permission
        r means read permission
        w means write permission
        x means execute permission
        --Note: in each group sequence of permission is always read write and execute.

      First group give information about owner permission
      Second group give information about group permission
      Third group give information about other user permission
    -- Second column give information about block.
        i.e. How many blocks are allocated to the file.

    -- Third column give information about file owner
        i.e. Who is owner of a file or who has created that file?


    -- Fourth column give information about group.
        i.e file belongs to which group?


    -- Fifth column give information about file size.


    -- Sixth column give information about file creation date or file modication date


    -- Seventh column give information about Name of a file.
        i.e What is name of a file or directory?


Example: ls command with option and with file name or directory name
Syntax :  ls option file_name

Example: ls -l f1.txt : It will display long information of f1.txt
