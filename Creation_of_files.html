Creation of files

    In Linux we can create files by using different command
    1. By using touch command
    2. By using cat command
    3. By using editors like (gedit,vi nano)


1.Creation of file using touch command.

    --- By using touch command we can create an empty file or we can create blank file.
    --- Syntax to create single empty file
                 touch file_name
           Example :     touch Abc.txt
           Ones command get executed then an empty file get created.
    --- Syntax to create multiple empty file at a time.
             touch file1_name file2_name file_name

    --- When we are creating multiple empty files at a time then file name should seperated by space.


When we should we go with touch command?

  1. If we want to create an empty file then we should go with touch command
  2. If we want to change time stamp of file then we should go with touch command
    Note: time stamp of a file means - date of creation or date of modifacation
