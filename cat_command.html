cat command

    cat stands for Concatenation (Concatenation means to join )
    With the help of cat command we can do following actions
    1. We can create a file.
    2. We can append data into existing file.
    3. We can merge multiple files into single file.
    4. we can display content of file on a screen.

How to Create File using catcommand:

    To Create file use below syntax
         cat>File_name
    cat>File_name
              type content here.
              after completion to save content and exist
              press   ctrl+D
    NOTE :- If named file is already available and has some content then all existing data get deleted

How to display file content on screeen

    Syntax:   cat -n file_name
    Example:  cat -n Abc.txt


How to append data into existing file.

    Syntax :  cat >> file_name
            type data here
            ones typing is completed to save and exist
            press ctrl+d

How to merge multiple files into single file.

    Syntax:   cat file1_name file2_name > destination_file_name
    Example:   cat file1_name file2_name > File12
