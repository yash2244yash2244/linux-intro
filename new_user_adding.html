How to create an user in Linux?

    To create an user we use command "adduser"
        syntax:   adduser group_name user_name
    -- Normal user can't add new user to the system
    -- to add new user to system you need admin rights
    -- first switch to root user and use below command.
         sudo adduser group_name user_name
    -- group name is an optional. If we don't provide group name then bydefault user name is considered as group name.
    -- user name must be in lower case.
    -- Ones user created that user information get stored inside
              /etc/passwd file.
