cut command

-- cut command is used to cut data from a file.
-- We can cut data from a file character by character
     or word by word
-- We can cut data based on seperator like
    colon :
    tild ~
    hash #
    percentage %
    space
    doller $
-- Syntax:
             cut option file_name
-- Options used in cut command
  1. -c : to cut data character by character

  2. -f: This option is used to specify field number
    Field number means column number

  3. -d : It is used to specify delimeter
    Delimeter means seperator

Note : -f and -d option always come in pair.

Case A: Cut data character by character
Note:First create a file as below


Example: Display only first character from every line of CutDemo.txt
Syntax:    cut -c 1 CutDemo.txt

Example: Display only fifth character from every line of Cutdemo.txt file
Syntax:   cut -c 5 Cutdemo.txt

Example: Display first 2 character from every line of cutdemo.txt file.
Syntax:   cut -c 1-2 cutdemo.txt

Note : to specify range of character we need to use hypen symbol

Example: Display first 5 character from every line of cutdemo.txt file.
Syntax:    cut -c 1-5 cutdemo.txt

Example: Display first character, third character and 10th character from every line of cutdemo.txt file
Syntax:    cut -c 1,3,10 cutdemo.txt

Example: Display first character, third character 10th and character from 11 to 15
from every line of cutdemo.txt file
Syntax:    cut -c 1,3,10,11-15 cutdemo.txt

B. Cut data based on seperator/delimeter


Note: First create a file


Example: seperator the data based on # and print first column
Syntax:   cut -d# -f1 Cutdemo2.txt


Example: Seperate the data based on # and print Third column
Syntax:   cut -d# -f3 cutdemo2.txt



Example: seperate the data based on # and print second column
Syntax:   cut -d# -f2 cutdemo2.txt
