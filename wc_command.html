wc command

    - wc stands for word count.
    - Purpose: wc command is used for below purpose
    1. count number lines
    2. count number of words
    3. count number of characters
    - Single space is considered as one characters
    - Syntax:  wc option file_name
    - Here option is optional part.
    - Different option of wc command are as below
    1. -l : To count number of lines
    2. -w : To count number of words
    3. -c : To count number of characters
    4. -L : To count number of characters present in in longest line

    -If we execute wc command without any option then in the out put
     we will get line count, word count and character count
                    wc file_name


    -If we want only line number count use -l option
                    wc -l file_name


    -If we want only word count then use -w option
                    wc -w file_name


    -If we want only character count the use -c option
                    wc -c file_name


    -If we want only count number of characters present in in longest line
                    wc -L file_name
