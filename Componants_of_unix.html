#Component of Unix

    Unix has two componants:-
    1. Shell
    2. Kernal


1.Shell

    - shell is outer layer of os.
    - shell read a command provided by user .
    - It cheak validness and correctness of command .
    - Here validness means command is available or not.
    - Here correctness means command is type in proper way or not.
    - If everything is proper then shell convert that command into kernal understandable
    -format and handover that command to kernal.
    - Shell act as interface between user and kernal.


2.Kernal

    - Kernal is core componant of Unix OS
    - Kernal is responsible to execute our command with the help of harware componant
    - memory allocation and processor allocation will take care by Kernal
    - Kernal act as interface between shell and harware componant
