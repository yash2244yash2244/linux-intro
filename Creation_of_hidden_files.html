Creation of hidden files and hidden directory.

    Q. What is mean by a hidden file?
    Ans:  Any file whose name start with dot is called as a hidden file.
    Example :
             Abc.txt   => Normal file
             .Abc.txt   => hidden file

    Q. What is mean by hidden directory?
    Ans: Any directory whose name start with dot is called as a hidden directory
    Example
             DirA   => Normal Directory
             .DirA  => hidden directory

    Q. Why we create a hidden file or hidden directory?
    Ans: To hide the data.
    Example: All files which are internaly used by Kernal are the are the hidden files.

How to create hidden file

  the command which are used to create normal file, same command are used to create hidden file.

    Example: create an empty hidden file.
    Ans:    touch file1.txt   = Normal file
            touch .file.txt   = hidden file

    Example: create multiple empty hidden files at a time
    Ans: touch .fileA.txt .fileB.txt .fileC.txt .fileD.txt

    Example: create hidden file using cat command.
    Ans:   cat > .file_name  
      type content here
      ones typing is completed
      To save the content and
      exit press ctrl+d

    Example: Append data to existiong hidden files
    Ans :   cat >> .file_name  
      type content here
      ones typing is completed
      press ctrl+d to save and exit.

Q. How to display hidden file content
Ans: The commands which are used to display
        normal file content, same command are used
        to display hidden file content.

Example: Display hidden file content using cat command
Ans:     cat -n hidden_file_name



Q. How to create a hidden directory?
Ans: The command which is used to create normal directory
          same command is used to create hidden directory

#Interconversion of hidden file and normal file

  1. Hidden to normal file
            mv hidden_file_name Normal_file_name

  2. Normal to hidden file
            mv Normal_file_name hidden_file_name

  3. Normal directory to hidden directory
            mv Normal_dir_name hidden_dir_name

  4. Hidden directory to normal directory
            mv hidden_dir_name Normal_dir_name
