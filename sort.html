Sort command


sort command:

    - Purpose : to sort file content
    - Syntax : sort option file_name
    - Here option is optional part (not mandatory)
    - By default sort command display file content in ascending order.
      For upper case A,B,C..Z
      For Lower case a,b,c..z
      For Digit 0,1,2,3..9


    - If we want to sort file content in descending order then use -r option


    - If file contains only upper case characters and lower case characters
       then file sorting file content system sort in below order
             a) Upper case characters b) Lower case characters


    - If file contains only numbers then these numbers get sorted based digit value


    -If file contains only number and we want to sort file content
      based on numeric value then use -n option


    - If file contains Alphanueric data (Lower case characters,Upper case characters and digit) and if we use -n option then sorting order is as below
      a. Upper case characters
      b. Lower case characters
      c. Digit


    -By default sort command display duplicate lines, If we want to print unique lines then use -u option
      Original File sort -u File
