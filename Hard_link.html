2.Hard link
    How to create hard link file?
    Ans:To create hard link we use 'ln' command without -s option.

    Syntax:   ln source_file_name hard_link_file_name

    Q. Why we create hard link?
    Ans: for back up purpose
    Important conclusion about hard link.
    1. Original file and hard link file have same time stamp
    2. Original file and hard link file have same size.
    3. Original file and hard link file have same inode number
    4. If we modify original file then same updation will reflect in hard link file
    5. If we modify hard link file then same updation will reflect in original file
    6. If we delete original file then there is no impact on hard link
    7. If we delete hard link file then there is no impact on original file.

      Note:we can create a hard link for file but not a directory
