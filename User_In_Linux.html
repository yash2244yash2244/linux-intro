User In Linux

    In Unix there are two types of Users
    1. Normal user = $prompt
    2. Root user/Admin user/Super user = #prompt


How to switch one user to another

    To switch one user to another user use below syntax
    syntax:- su - user_name

Here, su stand for switch user



How to switch Noraml User to Root User

    To switch Normal user to Root user use below syntax
    syntax:- sudo -i

Note:- Here, sudo stand for s=Super ,u=User ,do=Do
