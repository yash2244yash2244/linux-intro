Type of Software

    There are 3 types of software
    1. System Software
    2. Programming software
    3. Application software

1.System software:-

    This software is required to start and run the system.

    Example:

    1. all os software
    2. servers
    3. device drivers

2.Programming software:-

    This software is required to develope and run perticular language of program.

    Example:

    1. Suppose I want to develope and run a java program then It I need to install JDK (Java Developement Kit) software this JDK has compiler,debuggger,interpreter so compiler,debugger,interpreter come under the Programming Software

3.Application Software:-

    Based on the customer requirement some software are developed and these software are considered as Application software

    Example:

    1. Desktop application : calculator,paint,Ms office
    2. Banking application
    3. Mobile application : whatsapp,google pay,phone pay,paytm
