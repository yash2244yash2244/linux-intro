sed command:'Delete Action'


Action 2: Delete
    -- For delete action we use character 'd'
    Example: Delete line number 3 from sedDemo.txt file_name
           sed '3d' sedDemo.txt

    Note: Don't use -n option in delete action

    Example: Delete line number 5 and line number 10
           sed '5d;10d' sedDemo.txt

    Example; delete lines from 2 to 8
           sed '2,8d' sedDemo.txt

    Example: delete line number 2, line number 5 and lines from 8 to 11
           sed '2d;5d;8,11d' sedDemo.txt



Action 3: insert data before line

    -Syntax:  sed '3i typedata here' file_name
    Example: Insert data before line number 3 in sedDemo.txt file
        sed '3i THIS IS INSERTED DATA' file_name

Action 4: Add data after line

    -Syntax:   sed '3a typedata here' file_name
    Example: Add data after line number 3 in sedDemo.txt file
        sed '3a THIS IS INSERTED DATA AFTER LINE NUMBER 3 ' sedDemo.txt
