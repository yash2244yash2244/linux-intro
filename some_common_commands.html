1.pwd command
    - pwd stands for print working directory.
    - this command is used to print current working direct path
       pwd


2 ls command
    - ls stand for list out
    - this command is used to list out all files and directories
       ls


3 clear command
    - clear command is used to clear the terminal
      clear


4 date command
    date command is used to display today's date and time


5 cal command
    cal stand for calender
    cal command is used to display current month calender


6 cd command
    - cd stands for change directory
    - use: To navigate/move/switch/jump from one directory to another directory
    - Navigation is of two types
      1 Forword direction navigation
      2 Backword direction navigation

    ---- Forword direction Movement
    syntax
    cd directory_name

    Example: Suppose I am inside parent directory (/) and
    I want to move its subdirectory "bin" then
    Ans: cd /bin

    ---- Backword Directory movement
    syntax:
    cd .. (move one step back)


7. uptime
- uptime command is used to know how long system is running

- how to use uptime command?
  type 'uptime' and press enter

Ones command executed we will get below output information get display
    1) Current time
    2)up time (how long system is running)
    3)Number of logged users.
    4)Load average value for first minute, for fifth minute and for 15th minute


8. whoami
- whoami command is used to know your user name.
or
whoami command is used to know current logged user name
- How to use?
Ans: type 'whoami' and press enter

- Ones command get executed then in the output we will get user name.
Note: who command is also used to get user name.


9. hostname
- hostname command is used to know your system information
like system name,system ip address, network ip address.

- How to use?
Ans: Type 'hostname' and press enter.
Ones command get executed then in the output we will get system name.
If we want extra information like system ip address or network ip address
    -i :=> System IP address
    -I :=> Network IP address
    -f :=> System Name


10. uname
- uname command not used to get user related information
- uname command give operating system related information
- syntax uname option
i) -a : All information    uname -a ones this command get executed then we will get
below information in given order.
  1. a) Kernal name (use -s option)
  2. b) Hostname name (use -n option)
  3. c) Kernal release (use -r option)
  4. d) Kernal version (use -v option)
  5. e) Machine hardware name (use -m option)
  6. f) Processor type (use -p option)
  7. g) Hardware Platform (use -i option)
  8. h) Operating system name (use -o option)
