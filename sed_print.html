sed command:'Print' Action

Action 1: Print

    - For print action/operation we use 'p' character
      Example : print line number 1 from sedDemo.txt
             sed -n '1p' sedDemo.txt

      Note: while printing data using sed command we need to specify option -n
      If we don't specify -n option then target line get printed two times and
      other lines also get printed

      Example: print line number 5 form seddemo.txt
             sed -n '5p' sedDemo.txt

      Example: print header of a sedDemo.txt
             sed -n '1p' sedDemo.txt

      Example: print last line of seddemo.txt
             sed -n '$p' seddemo.txt

      Note: $ stands for last line

      Example: Print footer of sedDemo.txt
             sed -n '$p' seddemo.txt

      Example : Print line number 2, line number 5 and line number 7
             sed -n '2p;5p;7p' sedDemo.txt

      Note: If we want to print multiple lines then those line number should be
      seperated by semicolon

      Example: Print header, footer , line number 10th and line number 8
             sed -n '1p;$p;10p;8p' seddemo.txt

      Example: print lines from 5 to 10
             sed -n '5,10p' sedDemo.txt

      Note: To specify range of lines then use comma

      Example: print line number 2, line number 5 and lines from 7 to 11
             sed -n '2p;5p;7,11p' seddemo.txt

      Example: Print lines from 2 to 6 and 7 to 11
             sed -n '2,6p;7,11p' seddemo.txt
